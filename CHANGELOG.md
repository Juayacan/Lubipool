# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-23

### Added
- Initial release of LubiPool Card
- Tesla-style animated power flow diagram
- Real-time power consumption visualization
- Water quality score calculation (pH, ORP, temperature)
- Support for multiple equipment types:
  - Filtration pumps
  - Salt chlorinators
  - Heat pumps
  - Pool covers
  - pH regulators
  - UV systems
  - Ozone generators
  - Pool lights
- Adaptive flow animation based on power consumption
- Cost calculation in real-time
- Clickable entities for more information
- Fully configurable via YAML
- Responsive design (mobile and desktop)
- Visual editor for easy configuration
- Custom color and icon support per equipment
- Quality ranges customization
- Display modes: power and water flow
- TypeScript implementation with full typing
- Optimized bundle (27KB minified)

### Features
- Dynamic equipment positioning
- Automatic quality score calculation
- Runtime display for equipment
- Support for custom equipment colors
- Smooth SVG animations
- Theme-aware styling (light/dark mode)
- Configurable flow animation speeds
- Display zero power option per equipment
